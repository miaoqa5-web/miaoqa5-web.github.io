HTML 基础知识整理

Posted on 2026-01-25 Edited on 2026-02-28 In 基础笔记

### 一、基本结构

每一个网页都有着极其严谨的基础骨架：


<!DOCTYPE html>

<html>

<head>

&nbsp;   <title></title>

</head>

<body>

</body>

</html>



\### 二、常用标签



\* `<div>`：区块容器

\* `<p>`：段落

\* `<h1>` - `<h6>`：标题层级

\* `<img>`：图片

\* `<a>`：超链接

\* `<ul>` / `<li>`：无序列表与列表项



\### 三、语义化标签



现代网页开发提倡使用具备特定含义的标签，告别满篇的 `<div>`：

\* `<header>`：头部区域

\* `<section>`：独立的内容区块

\* `<article>`：完整的文章主体

\* `<footer>`：底部版权或参考区域



> \*\*核心心得：\*\* 语义化标签能够极大提升网页代码的可读性与结构清晰度，让网页本身具备良好的信息架构。



\### 四、总结

HTML 本质上是一种\*\*树状结构\*\*。理解标签之间的\*\*嵌套关系\*\*（就像梳理论文的逻辑大纲一样），是后续进行 CSS 布局和精准定位的基础。
