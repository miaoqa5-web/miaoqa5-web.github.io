CSS3 盒模型与 Flexbox 现代布局

Posted on 2026-02-23 Edited on 2026-02-28 In \[Web Development]

HTML 搭建了网页的骨架，而 CSS（层叠样式表）则赋予了网页皮囊。为了让我的 Hexo 博客摆脱默认主题的千篇一律，我重点攻克了 CSS 的两大核心概念：盒模型与 Flex 布局。

1. 深入理解盒模型（Box Model）

网页上的每一个元素，在浏览器看来都是一个矩形的“盒子”。理解这个概念，是解决所有网页排版错位问题的基础。

     

  • Content（内容）：文字或图片本身的区域。

    •  

  • Padding（内边距）：内容到边框的距离（可以理解为画框里的留白）。

    •  

  • Border（边框）：盒子的边缘。

    •  

  • Margin（外边距）：盒子与其他盒子之间的距离（可以理解为画框与画框之间的墙壁距离）。

2. 告别传统排版：Flexbox 弹性盒子

以前的网页布局为了让几个元素并排，需要用到繁琐的 float（浮动）。而Flexbox 提供了一种优雅的二维布局方式。只需给父元素加上 display: flex; ，内部子元素就能轻松实现居中、等距排列。

3. 实践案例：设计独立音乐卡片

为了巩固所学，我利用 CSS3 的 border-radius（圆角）和 box-shadow（阴影）属性，结合 Flex 布局，写了一个唱片信息展示卡片。



/\* 卡片容器：使用 Flex 布局居中对齐 \*/

.quote-card {

    display: flex;

    align-items: center;

    background-color: #f9f9f9;

    border-left: 4px solid #d32f2f;

    padding: 20px;

    margin: 20px 0;

    box-shadow: 0 4px 8px rgba(0,0,0,0.05);

    border-radius: 8px;

    transition: transform 0.3s ease;

}



/\* 增加鼠标悬停的微交互动效 \*/

.quote-card:hover {

    transform: translateY(-5px);

    box-shadow: 0 8px 16px rgba(0,0,0,0.1);

}

学习心得：
通过编写 CSS这种底层样式的微调，比直接使用 Hexo 插件更加锻炼我对页面逻辑的理解。
