CSS 基础知识

Posted on 2026-02-28

## 一、CSS 核心概念

1. CSS 定义：层叠样式表（Cascading Style Sheets），用于控制HTML文档的呈现样式（外观、布局、行为），实现结构（HTML）与样式（CSS）分离。

2. 核心作用：

  - 美化网页：设置颜色、字体、间距、背景等视觉样式

  - 布局控制：实现元素的排列、定位、响应式适配

  - 交互增强：配合伪类/伪元素实现基础交互效果

3. CSS 版本：

  - CSS1：基础样式（字体、颜色、边距）

  - CSS2：引入浮动、定位、媒体查询、伪类

  - CSS3：模块化升级（圆角、阴影、渐变、动画、弹性盒、网格布局），目前主流使用CSS3特性

## 二、CSS 引入方式

### 1. 行内样式（内联样式）

- 语法：直接在HTML标签的style属性中编写CSS

- 示例：<div style="width: 100px; height: 100px; background: red;"></div>

- 特点：优先级最高，仅作用于当前标签，不利于维护，仅临时调试使用

### 2. 内部样式（嵌入式）

- 语法：在HTML的<head>标签内通过<style>标签编写CSS

- 特点：仅作用于当前HTML文件，适合单页面样式管理

### 3. 外部样式（链接式）

- 语法：通过<link>标签引入外部.css文件

- 示例：<link rel="stylesheet" href="style.css">

- 特点：可复用，利于多页面样式统一管理，是项目开发的主流方式

- 补充：还可通过@import引入（如@import url("style.css");），但优先级低于<link>，加载性能稍差

## 三、CSS 基本语法

1. 语法结构：选择器 + 声明块

  - 选择器：指定要样式化的HTML元素

  - 声明块：由{}包裹，内部是多个声明（属性: 值;）

2. 语法规则：

  - 声明必须以;结尾（最后一个声明可省略，但建议统一添加）

  - 属性与值之间用:分隔

  - 大小写不敏感，但建议属性小写、值按规范（如颜色值）

  - 注释：单行/多行均用/\* 注释内容 \*/（无//注释）

## 四、CSS 选择器（核心）

### 1. 基础选择器

- 标签选择器：匹配指定标签名的所有元素

  语法：标签名 { … }

  示例：p { color: black; }（所有

标签文字黑色）

- 类选择器：匹配指定class属性的元素

  语法：.类名 { … }

  示例：.box { width: 100px; }（所有class=”box”的元素宽度100px）

  补充：一个元素可拥有多个类，如<div class="box red"></div>

- ID选择器：匹配指定id属性的元素（页面唯一）

  语法：#id名 { … }

  示例：#header { height: 80px; }（id=”header”的元素高度80px）

- 通配符选择器：匹配所有元素

  语法：* { … }

  示例：\* { margin: 0; padding: 0; }（清除所有元素的默认边距）

  注意：性能较差，尽量少用

### 2. 复合选择器

- 后代选择器：匹配指定元素的所有后代元素（包含子、孙等）

  语法：选择器1 选择器2 { … }

  示例：ul li { list-style: none; }（ul下所有li去掉列表样式）

- 子选择器：仅匹配指定元素的直接子元素

  语法：选择器1 > 选择器2 { … }

  示例：ul > li { color: blue; }（仅ul直接子li文字蓝色，孙li不生效）

- 并集选择器：匹配多个选择器的元素

  语法：选择器1, 选择器2 { … }

  示例：h1, h2, p { margin: 10px 0; }（h1、h2、p都设置上下边距）

- 交集选择器：匹配同时满足多个选择器的元素

  语法：选择器1选择器2 { … }（无空格）

  示例：p.red { color: red; }（class=”red”的p标签文字红色）

- 相邻兄弟选择器：匹配指定元素的下一个相邻兄弟元素

  语法：选择器1 + 选择器2 { … }

  示例：h1 + p { font-size: 16px; }（h1紧跟的第一个p标签字号16px）

- 通用兄弟选择器：匹配指定元素的所有后续兄弟元素

  语法：选择器1 ~ 选择器2 { … }

  示例：h1 ~ p { color: gray; }（h1之后所有同级p标签文字灰色）

### 3. 属性选择器

- 匹配拥有指定属性的元素：\[attr] { ... }

  示例：\[title] { cursor: pointer; }（有title属性的元素鼠标变指针）

- 匹配属性等于指定值的元素：\[attr=value] { ... }

  示例：input\[type="text"] { border: 1px solid #ccc; }（type为text的input边框样式）

- 匹配属性以指定值开头的元素：\[attr^=value] { ... }

  示例：a\[href^="http"] { color: blue; }（href以http开头的a标签蓝色）

- 匹配属性以指定值结尾的元素：\[attr$=value] { ... }

  示例：a\[href$=".pdf"] { color: red; }（href以.pdf结尾的a标签红色）

- 匹配属性包含指定值的元素：\[attr\*=value] { ... }

  示例：a\[href\*="baidu"] { font-weight: bold; }（href包含baidu的a标签加粗）

### 4. 伪类选择器（状态选择器）

- 链接伪类（需按顺序：LVHA）：

  - :link：未访问的链接

  - :visited：已访问的链接

  - :hover：鼠标悬停在元素上

  - :active：元素被激活（点击时）

  示例：a:hover { color: orange; }（鼠标悬停链接变橙色）

- 表单伪类：

  - :focus：元素获得焦点（如input输入时）

  - :enabled：可用的表单元素

  - :disabled：禁用的表单元素

  - :checked：选中的单选/复选框

  示例：input:focus { outline: none; border: 1px solid blue; }（input获焦时去掉默认轮廓，边框变蓝）

- 结构伪类：

  - :first-child：父元素的第一个子元素

  - :last-child：父元素的最后一个子元素

  - :nth-child(n)：父元素的第n个子元素（n可为数字、odd/even、公式如2n）

  - :first-of-type：父元素中指定类型的第一个元素

  - :last-of-type：父元素中指定类型的最后一个元素

  - :nth-of-type(n)：父元素中指定类型的第n个元素

  示例：ul li:nth-child(odd) { background: #f5f5f5; }（ul下奇数行li背景灰色）

- 其他伪类：

  - :not(选择器)：排除指定选择器的元素

  示例：li:not(.active) { color: #666; }（排除class=active的li，其余文字灰色）

### 5. 伪元素选择器（创建虚拟元素）

- ::before：在元素内容前插入虚拟内容

- ::after：在元素内容后插入虚拟内容

- ::first-letter：匹配文本的第一个字符

- ::first-line：匹配文本的第一行

- 注意：伪元素必须配合content属性使用（空内容用content: “”）

  示例：

  ```

  .box::before {

  content: “前缀：”;

  color: red;

  }

  .text::first-line {

  font-size: 18px;

  font-weight: bold;

  }

  ```

### 6. 选择器优先级

- 优先级规则（从高到低）：

  1. !important（最高，慎用）

  2. 行内样式（style属性）

  3. ID选择器（#id）

  4. 类选择器（.class）、属性选择器、伪类选择器

  5. 标签选择器、伪元素选择器

  6. 通配符选择器（*）、继承的样式

- 优先级计算（权重值）：

  - 行内样式：1,0,0,0

  - ID选择器：0,1,0,0

  - 类/属性/伪类：0,0,1,0

  - 标签/伪元素：0,0,0,1

  - 通配符：0,0,0,0

  - 示例：#header .nav li 权重为 0,1,1,1；div.box 权重为 0,0,1,1

- 层叠规则：优先级高的样式覆盖低的；优先级相同则后写的样式覆盖先写的

## 五、CSS 常用样式属性

### 1. 字体样式（Font）

- font-family：字体家族（多个字体用,分隔，最后写通用字体）

  示例：font-family: "Microsoft Yahei", Arial, sans-serif;

- font-size：字体大小（单位：px/em/rem/%）

  示例：font-size: 16px;（默认16px）

- font-weight：字体粗细（normal/400、bold/700、100-900）

  示例：font-weight: 600;

- font-style：字体样式（normal/italic/oblique）

  示例：font-style: italic;（斜体）

- font：复合属性（顺序：style weight size family）

  示例：font: italic bold 16px "Microsoft Yahei";

### 2. 文本样式（Text）

- color：文本颜色（十六进制#fff、rgb(255,255,255)、rgba(255,255,255,0.5)、英文red）

- text-align：文本对齐（left/center/right/justify）

- text-decoration：文本装饰（none/underline/overline/line-through）

  示例：a { text-decoration: none; }（去掉链接下划线）

- text-indent：文本缩进（单位：px/em）

  示例：p { text-indent: 2em; }（首行缩进2字符）

- line-height：行高（单位：px/em/无单位、%）

  示例：p { line-height: 1.5; }（行高为字号的1.5倍，推荐无单位）

- letter-spacing：字符间距

- word-spacing：单词间距（针对英文）

- text-transform：文本大小写（none/uppercase/lowercase/capitalize）

- white-space：空白处理（normal/nowrap/pre）

  示例：div { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; }（文本超出省略）

- text-overflow：文本溢出处理（clip/ellipsis）

### 3. 背景样式（Background）

- background-color：背景颜色

- background-image：背景图片

  示例：background-image: url("bg.jpg");

- background-repeat：背景重复（repeat/repeat-x/repeat-y/no-repeat）

- background-position：背景位置（x y，可填像素、百分比、方位词left/center/right top/center/bottom）

  示例：background-position: center top;

- background-size：背景尺寸（auto/cover/contain/具体值）

  示例：background-size: cover;（覆盖容器，可能裁剪）

- background-attachment：背景固定（scroll/fixed）

- background：复合属性（顺序：color image repeat position size attachment）

  示例：background: #f5f5f5 url("bg.jpg") no-repeat center/cover fixed;

### 4. 盒模型（核心）

- 盒模型组成（从内到外）：content → padding → border → margin

- width/height：默认指内容区宽度/高度（可通过box-sizing修改）

- padding：内边距（元素内容与边框的距离）

  语法：padding: 上 右 下 左；padding-top/padding-right/padding-bottom/padding-left

  示例：padding: 10px 20px;（上下10px，左右20px）

- border：边框（复合属性：width style color）

  语法：border: 1px solid #ccc;

  细分：border-width/border-style（solid/dashed/dotted/double）/border-color

  单边：border-top: 1px solid red;

- margin：外边距（元素与其他元素的距离）

  语法：同padding，margin-top/margin-right/margin-bottom/margin-left

  注意：垂直方向margin会重叠（塌陷），水平方向不会；margin: 0 auto可实现块级元素水平居中

- box-sizing：盒模型类型

  - content-box（默认）：width/height仅包含content

  - border-box：width/height包含content+padding+border（推荐，方便布局）

  示例：\* { box-sizing: border-box; }

- display：元素显示类型

  - block：块级元素（独占一行，可设宽高，如div/h1/p）

  - inline：行内元素（不独占一行，不可设宽高，如span/a/em）

  - inline-block：行内块元素（不独占一行，可设宽高，如img/input）

  - none：隐藏元素（不占位置）

  - flex：弹性盒容器

  - grid：网格容器

- visibility：可见性（visible/hidden，hidden时元素仍占位置）

- overflow：内容溢出处理

  - visible：默认，溢出显示

  - hidden：溢出隐藏

  - scroll：显示滚动条（无论是否溢出）

  - auto：溢出时显示滚动条

### 5. 定位（Position）

- position：定位类型

  - static：默认，无定位（不配合top/right/bottom/left使用）

  - relative：相对定位（相对于自身原位置偏移，仍占原位置）

  - absolute：绝对定位（相对于最近的已定位祖先元素偏移，不占原位置）

  - fixed：固定定位（相对于浏览器视口偏移，不占原位置，滚动时位置不变）

  - sticky：粘性定位（滚动到指定位置时从relative变为fixed）

- 偏移属性：top/right/bottom/left（配合定位使用，单位px/%）

- z-index：层级（数值越大层级越高，仅对定位元素生效）

  示例：

  ```

  .box {

  position: relative;

  top: 10px;

  left: 20px;

  z-index: 10;

  }

  .popup {

  position: absolute;

  top: 50%;

  left: 50%;

  transform: translate(-50%, -50%); /* 居中 */

  z-index: 100;

  }

  ```

### 6. 浮动（Float）

- float：浮动方向（left/right/none）

- 作用：让块级元素并排显示（早期布局核心）

- 清除浮动（解决父元素高度塌陷）：

  方法1：父元素添加overflow: hidden;

  方法2：在浮动元素后加空标签，设置clear: both;

  方法3：伪元素清除（推荐）：

  ```

  .clearfix::after {

  content: “”;

  display: block;

  clear: both;

  visibility: hidden;

  height: 0;

  }

  .clearfix { zoom: 1; } /* 兼容IE6/7 */

  ```

- clear：清除浮动（left/right/both）

### 7. 弹性盒布局（Flex，主流）

- 核心：给父元素设置display: flex;，子元素变为flex项

- 父元素属性：

  - flex-direction：主轴方向（row/row-reverse/column/column-reverse）

  - justify-content：主轴对齐方式（flex-start/flex-end/center/space-between/space-around/space-evenly）

  - align-items：侧轴对齐方式（flex-start/flex-end/center/baseline/stretch）

  - flex-wrap：是否换行（nowrap/wrap/wrap-reverse）

  - align-content：多行侧轴对齐（仅换行时生效，同justify-content值）

  - flex-flow：复合属性（flex-direction + flex-wrap）

- 子元素属性：

  - flex-grow：放大比例（默认0，剩余空间分配）

  - flex-shrink：缩小比例（默认1，空间不足时缩小）

  - flex-basis：默认尺寸（auto/px/%）

  - flex：复合属性（grow shrink basis，推荐用flex: 1; 等分）

  - align-self：单独设置自身侧轴对齐（覆盖align-items）

  - order：排列顺序（数值越小越靠前，默认0）

- 示例（水平垂直居中）：

  ```

  .container {

  display: flex;

  justify-content: center;

  align-items: center;

  width: 300px;

  height: 300px;

  border: 1px solid #ccc;

  }

  .item {

  width: 100px;

  height: 100px;

  background: red;

  }

  ```

### 8. 网格布局（Grid）

- 核心：二维布局（行+列），比Flex更适合复杂布局

- 父元素属性：

  - display: grid;：设为网格容器

  - grid-template-columns：列宽（repeat(3, 100px) 表示3列，每列100px）

  - grid-template-rows：行高（同列宽）

  - grid-gap/gap：网格间距（row-gap + column-gap）

  - justify-items：单元格内容水平对齐

  - align-items：单元格内容垂直对齐

  - grid-template-areas：网格区域命名

- 子元素属性：

  - grid-column：列跨度（grid-column: 1 / 3; 跨1-3列）

  - grid-row：行跨度

  - grid-area：指定区域/行列跨度

- 示例：

  ```

  .grid-container {

  display: grid;

  grid-template-columns: repeat(3, 1fr); /* 3列等分 */

  grid-gap: 10px;

  width: 300px;

  height: 300px;

  }

  .grid-item {

  background: blue;

  color: white;

  text-align: center;

  line-height: 90px;

  }

  ```

### 9. 过渡与动画

- 过渡（Transition）：元素状态变化时的平滑过渡

  语法：transition: 属性 时长 过渡函数 延迟;

  示例：

  ```

  .box {

  width: 100px;

  height: 100px;

  background: red;

  transition: all 0.3s ease;

  }

  .box:hover {

  width: 200px;

  background: blue;

  }

  ```

- 动画（Animation）：自定义关键帧动画

  步骤：

  1. 定义关键帧：@keyframes 动画名 { 0% { ... } 50% { ... } 100% { ... } }

  2. 应用动画：animation: 动画名 时长 速度曲线 延迟 播放次数 是否反向;

  示例：

  ```

  @keyframes move {

  0% { transform: translateX(0); }

  100% { transform: translateX(200px); }

  }

  .box {

  width: 100px;

  height: 100px;

  background: red;

  animation: move 2s ease infinite alternate;

  }

  ```

- transform：变换（不影响布局）

  - 平移：translate(x, y) / translateX(x) / translateY(y)

  - 旋转：rotate(deg)

  - 缩放：scale(x, y) / scaleX(x) / scaleY(y)

  - 倾斜：skew(deg, deg)

### 10. 响应式布局

- 核心：根据不同屏幕尺寸适配样式

- 媒体查询（Media Query）：

  语法：@media 媒体类型 and (条件) { 样式 }

  媒体类型：screen（屏幕）、print（打印）、all（所有）

  条件：width、height、orientation（portrait/landscape）

  示例：

  ```

  /* 屏幕宽度小于768px时生效 */

  @media screen and (max-width: 768px) {

  .container {

  width: 100%;

  padding: 0 10px;

  }

  }

  /* 屏幕宽度大于1200px时生效 */

  @media screen and (min-width: 1200px) {

  .container {

  width: 1200px;

  margin: 0 auto;

  }

  }

  ```

- 视口设置（必须加）：

  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  - width=device-width：视口宽度等于设备宽度

  - initial-scale=1.0：初始缩放比例1:1

## 六、CSS 其他重要知识点

### 1. 单位

- 绝对单位：px（像素）、cm（厘米）、mm（毫米）、in（英寸）、pt（点）

- 相对单位：

  - em：相对于父元素字体大小（如父元素16px，1em=16px）

  - rem：相对于根元素（html）字体大小（推荐，避免嵌套问题）

  - %：相对于父元素对应属性值

  - vw：视口宽度的1%（1vw=设备宽度/100）

  - vh：视口高度的1%

  - vmin：vw和vh中较小值

  - vmax：vw和vh中较大值

- 推荐用法：

  - 字体大小：rem

  - 布局宽度/高度：px/%/vw/vh

  - 行高：无单位（如1.5）

### 2. 继承性

- 可继承属性：字体、文本、颜色相关（font-*/text-*/color/line-height）

- 不可继承属性：盒模型、定位、背景相关（width/height/margin/padding/border/position/background）

- 强制继承：属性: inherit;（如border: inherit;）

### 3. 浏览器前缀（兼容旧浏览器）

- 主流前缀：

  - -webkit-（Chrome/Safari）

  - -moz-（Firefox）

  - -ms-（IE）

  - -o-（Opera）

- 示例：

  ```

  .box {

  -webkit-border-radius: 5px;

  -moz-border-radius: 5px;

  border-radius: 5px;

  }

  ```

### 4. 常见重置样式（Normalize.css/Reset.css）

- 目的：清除浏览器默认样式，统一跨浏览器表现

- 基础重置：

  ```

  * {

  margin: 0;

  padding: 0;

  box-sizing: border-box;

  }

  li {

  list-style: none;

  }

  a {

  text-decoration: none;

  color: inherit;

  }

  img {
