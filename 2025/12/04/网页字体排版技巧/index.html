Markdown 排版与样式指南

Posted on 2025-12-04 Edited on 2025-12-05 In Notes

Hexo 博客中常用的 Markdown 排版技巧，复制源码即可使用。

1. 文字样式

  • 加粗: 用两个*包围内容，**我是粗体** → 我是粗体
  • 斜体: 用一个*包围内容，*我是斜体* → 我是斜体
  • 删除线:用两个~包围内容， ~~被删除的文字~~ → 被删除的文字
  • 行内代码: 用反引号 ` 包裹，例如 .`熊敲代码`.

2. 章节标题

在文字前加上两个#，即 ##

3. 引用与强调

这是一个引用块 (Blockquote)。
适合用来放名言、摘录或者特别说明。

语法：在行首加 > 符号。

4. 列表

无序列表 (使用 * 或 - 加空格):

  • 观点一
  • 观点二

语法： *+3个空格

  • 观点一
    • 子观点 A (行首缩进2个空格)

语法： 3个空格+*+3个空格

有序列表 (使用数字加点):

  1. 第一步：打开冰箱
  2. 第二步：把大象放进去

5. 插入图片

语法：![图片描述](图片路径)

建议将图片放在 source/images/ 文件夹下。

6. 插入链接

语法：[内容](https://链接)
如[我的 GitHub 主页](https://github.com/miaoqa5-web)

  • 我的 GitHub 主页
  • 发邮件给我

7. 表格 (Table)

语法：用 | 分隔列，用 --- 分隔表头。
如：| 网站名称 | 类型 | 评价 |

| :— | :— | :— |

| Pitchfork | 乐评 | ⭐⭐⭐⭐ |

| RYM | 数据库 | ⭐⭐⭐⭐⭐ |

网站名称 类型 评价
Pitchfork 乐评 ⭐⭐⭐⭐
RYM 数据库 ⭐⭐⭐⭐⭐

8. 代码块 (Code Block)

用来展示代码，支持语法高亮。
语法：使用三个反引号包围代码，即.```...```.
