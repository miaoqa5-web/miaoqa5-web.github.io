常用命令备忘录

Posted on 2025-12-04 In Notes

1. 进入博客目录

cd /d/hexo-blog/blog

2. 本地预览网站

hexo clean
hexo g
hexo s
简写：hexo s -g

访问浏览器：http://localhost:4000

3. 部署到GitHub Pages

hexo clean
hexo g
hexo d
简写：hexo g -d

4. 创建新文章

hexo new post “文章标题”
文件位置：source/_posts/文章标题.md

5. 加标签类别
